## 📝 Qué me falta hacer?

Aplicación web interactiva de lista de tareas desarrollada con JavaScript, HTML y CSS, que permite gestionar tareas de forma dinámica, persistente y visual.

---
## 🚀 Funcionalidades
- ➕ Agregar nuevas tareas
- ❌ Eliminar tareas
- ✅ Marcar tareas como completadas
- 💾 Persistencia de datos con localStorage
- 🚫 Validación para evitar tareas duplicadas
- 🎉 Animación de confetti al completar tareas
- 🎨 Interfaz mejorada con Bootstrap y CSS personalizado

---
## Utilicé:
- HTML5
- CSS3
- JavaScript
- Bootstrap

---
## 🎯 Objetivos del proyecto

Este proyecto fue desarrollado como trabajo práctico con el objetivo de:

- Manipular el DOM dinámicamente
- Gestionar eventos de usuario
- Trabajar con arrays y objetos en JavaScript
- Implementar persistencia con localStorage
- Integrar mejoras visuales en una interfaz web
